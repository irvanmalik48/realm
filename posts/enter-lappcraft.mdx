## Introduction

Want to join my Minecraft server? This guide will walk you through the process of setting up your launcher and account to get started.

> **Note for Advanced Users:** If you are already using a launcher that supports `.mrpack` imports (like Prism Launcher, ElyPrismLauncher, or PolyMC) and have a paid Mojang or Ely.by account, you can skip directly to [Step 3: Import the Modpack](#step-3-import-the-modpack). This guide focuses on setting up Fjord Launcher with a Drasl account.

## Step 1: Download Fjord Launcher

First, you'll need the Fjord Launcher. It's a custom launcher that supports our authentication system.

1. Go to the [Fjord Launcher GitHub Releases](https://github.com/unmojang/FjordLauncher).
2. Download the installer for your operating system:
   - **Windows**: Download the `.exe` file or the windows installer.
   - **Linux**: Download the `.AppImage`, `.deb`, or `.rpm` file depending on your distribution. Also available in AUR if you use Arch Linux or its derivatives.
3. Install and run the launcher on your system.

## Step 2: Set Up Your Account

We use a custom authentication server (Drasl) for Lappcraft, but you can also use a paid Mojang or Ely.by account. If you choose Drasl, you need to create an account there first.

1. Go to the [Drasl Registration Page](https://drasl.irvanma.eu.org/web/registration).
2. Fill in the **Username** and **Password** fields.
3. (Optional) You can leave the **Player UUID** field blank to have one generated for you.
4. Click **Register**.

Once registered, you can add your account to the launcher.

> **Note:** If you don't have a paid Minecraft Java Edition account, you can use [this offline bypass trick](https://github.com/antunnitraj/Prism-Launcher-PolyMC-Offline-Bypass/tree/main) before adding your account.

1. Open Fjord Launcher.
2. Go to **Settings** > **Accounts**.
3. Click **Add authlib-injector** on the right side.
4. In the dialog that appears:
   - **Email/username**: Enter your username.
   - **Password**: Enter your password.
   - **API Server URL**: Enter `https://drasl.irvanma.eu.org/authlib-injector`.
5. Click **OK** to add your account.

## Step 3: Import the Modpack

We have a custom modpack for the server. You can import it directly into Fjord Launcher via URL.

1. Copy this URL: `https://storage.irvanma.eu.org/Lappcraft4.mrpack`
2. In Fjord Launcher, click the **Add Instance** button on top left.
3. Select the **Import** tab on the left side.
4. Paste the URL into the "Local file or link to a direct download" field.
5. Give the instance a name (e.g., "Lappcraft").
6. Click **OK** and wait for the launcher to download the necessary files.

## Step 4: Ready to Go!

Once the installation is complete:

1. Select the new Lappcraft instance in the launcher.
2. Click **Launch** button on the right sidebar.
3. You are now ready to join the world!

See you on the server!
