## Math Equations (KaTeX)

We now support LaTeX/KaTeX math equations! 

### Inline Math
The classic Einstein equation is $E = mc^2$. You can also write fractions like $\frac{a}{b}$ inline within your text.

### Block Math
For larger, standalone equations, display block math can be used:

$$
f(x) = \int_{-\infty}^{\infty} \hat{f}(\xi)\,e^{2 \pi i \xi x}\,d\xi
$$

Another example showing summations:

$$
\sum_{i=1}^{n} i = \frac{n(n+1)}{2}
$$

## Code Blocks with Copy Button

Here is a code block showcasing custom file title rendering (`remark-flexible-code-titles`) and our new copy-to-clipboard button:

```javascript:Fibonacci.js
// Function to generate the Fibonacci sequence up to N terms
function generateFibonacci(n) {
  let seq = [0, 1];
  for (let i = seq.length; i < n; i++) {
    seq.push(seq[i - 1] + seq[i - 2]);
  }
  return seq;
}

console.log(generateFibonacci(10));
```
